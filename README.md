# docker проект: генератор и аналитик данных о книгах

## описание проекта

Все состоит из двух docker-контейнеров:
1) **генератор данных** - создает CSV файл с данными о книгах
2) **аналитик данных** - читает CSV и создает отчет со статистикой(и сервак можно запустить)

данные хранятся на хосте и монтируются в контейнеры.

##  как запустить и работать

1) Склонируйте репозиторий, выполните `chmod +x run.sh`
2) затем последовательно запустите
`./run.sh build_generator`
`./run.sh build_reporter`
`./run.sh run_generator`
`./run.sh run_reporter` 
ЭТО для генерации данных и отчета
3) для просмотра `report.html` в браузере через GitHub Codespaces используйте
`./run.sh report_server` 
и откройте url с портом 8080 из вкладки ports, добавив `/report.html`
